СИСТЕМА ПРОГНОЗИРОВАНИЯ ЦЕН НА ПРОДУКТЫ ПИТАНИЯ

1. Общее описание проекта
Данный проект представляет собой учебную информационную систему прогнозирования цен
на продукты питания на основе методов машинного обучения. Система предназначена
для анализа исторических данных, обучения нейросетевой модели и формирования
краткосрочного прогноза цен с последующей визуализацией результатов.

Проект реализован в рамках дисциплины, связанной с архитектурой и разработкой
информационных систем, и включает несколько лабораторных работ (ЛР1–ЛР4),
каждая из которых отражает отдельный этап жизненного цикла системы

2. Функциональные возможности
Система обеспечивает следующие функции:
– загрузка и хранение исходных данных о ценах на продукты питания;
– преобразование месячных данных в дневной временной ряд;
– формирование обучающих выборок для нейросетевой модели;
– обучение рекуррентной нейронной сети LSTM с настраиваемым числом эпох;
– прогнозирование цен на фиксированный горизонт (30 дней);
– просмотр прогнозных значений для любой выбранной даты;
– расчёт метрик качества прогноза (MAE, RMSE);
– визуализация истории, прогноза и ошибок;
– экспорт результатов прогнозирования в CSV-файл;
– сохранение обученной модели в файл

3. Архитектура системы
Система реализована в виде монолитного приложения с логическим разделением
на функциональные модули. Основные компоненты системы:
– пользовательский интерфейс (Streamlit);
– модуль загрузки и подготовки данных;
– модуль формирования обучающих последовательностей;
– модуль обучения нейросетевой модели;
– модуль прогнозирования временных рядов;
– модуль оценки качества и визуализации результатов

Архитектурная схема системы приведена в лабораторной работе ЛР1
(диаграмма компонентов).

4. Используемые технологии и библиотеки
Язык программирования:
– Python 3.10+
Основные библиотеки:
– pandas — обработка и анализ данных;
– numpy — численные вычисления;

– TensorFlow / Keras — реализация и обучение нейросетевой модели LSTM;

6. Подготовка и запуск проекта
1. Установить Python версии 3.10 или выше.
2. Установить необходимые зависимости:
   pip install pandas numpy matplotlib scikit-learn tensorflow streamlit

3. Запустить приложение Streamlit:
   streamlit run app.py

После запуска веб-интерфейс будет доступен в браузере

7. Описание модели машинного обучения
В системе используется рекуррентная нейронная сеть типа LSTM
(Long Short-Term Memory), предназначенная для работы с временными рядами

Вход модели:
– последовательность значений временного ряда длиной lookback

Выход модели:
– прогноз значения цены на следующий временной шаг

Обучение модели проводится на временных данных без перемешивания.
В лабораторной работе ЛР4 модель обучалась на 10 000 эпох
для выполнения требований по экспериментальному исследованию

8. Метрики качества
Для оценки качества прогноза используются следующие метрики:
– MAE (Mean Absolute Error);
– RMSE (Root Mean Squared Error).

Также реализовано сравнение с baseline-моделью,
основанной на последнем наблюдаемом значении временного ряда

9. Экспериментальные результаты
Результаты обучения и прогнозирования приведены в лабораторной работе ЛР4
в виде таблиц и графиков, включая:
– кривые обучения (loss);
– сравнение метрик качества;
– визуализацию прогнозируемых значений;
– анализ времени отклика системы

10. Назначение лабораторных работ
ЛР1 — архитектура информационной системы и обоснование технических решений
ЛР2 — анализ и подготовка данных для задачи прогнозирования
ЛР3 — реализация работающего программного сервиса
ЛР4 — техническая статья с численными результатами и визуализацией

11. Ограничения и примечания
Проект носит учебный характер. Используемый набор данных ограничен
по объёму, что может влиять на качество прогноза и сравнительную
эффективность нейросетевой модели по отношению к простым baseline-подходам
